# The FlexWord Word List

This list powers FlexWord, a word game originally built for Reddit
You can play it at r/FlexWord.
A mobile version will be released soon.

I make this list public on purpose. There's no committee deciding what counts as a real
word. It's just me a forklift driver who taught himself how to build a word game.

I need your help keeping it honest.

## Word debates belong on Reddit

Got a word to suggest? Think something shouldn't be in here? There are two paths:
1. Open an issue on GitHub. I'll review it.
1. Head to r/FlexWord and use the **Word List** flair. That's where the conversation
happens.

## About the list

Forked from [Wordnik's open-source wordlist](https://github.com/wordnik/wordlist).
Wordnik is a nonprofit dictionary project. If you find their work useful,
they accept donations.

The list has been curated for use as a game word list — which is a different
bar than a dictionary. Words that exist in English aren't automatically
appropriate as puzzle answers. Ongoing curation is logged in CHANGELOG.md.
